# EMPLOYEE_LEAVE_MANAGEMENT_SYSTEM
Employee Leave Management System (ELMS) is a Java-based web application that allows employees to apply for leaves and track balances, while admins manage employees and approve or reject requests. Built using JSP, Servlets, JDBC, and MySQL with a clean UI and secure session handling.
